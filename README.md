# Email Enhancer - Браузерное расширение для улучшения работы с электронной почтой

## 📧 Описание

Email Enhancer - это мощное браузерное расширение, которое значительно улучшает опыт работы с электронной почтой. Расширение предоставляет инструменты для голосового ввода, автоматического исправления ошибок, AI-улучшения текста и поддерживает множество языков.

## ✨ Основные функции

### 🎤 Голосовой ввод
- Запись голосовых сообщений
- Автоматическое преобразование речи в текст
- Поддержка 6 языков распознавания

### ✅ Исправление текста
- Автоматическое исправление орфографических ошибок
- Коррекция грамматических ошибок
- Улучшение пунктуации и форматирования

### 🤖 AI-улучшение текста
- **Дружелюбный стиль** - для неформального общения
- **Официальный стиль** - для деловой переписки
- **Краткий стиль** - для лаконичных сообщений
- **Убеждающий стиль** - для маркетинговых писем
- **Технический стиль** - для технической документации

### 🌍 Многоязычность
Поддерживаемые языки:
- 🇷🇺 Русский
- 🇺🇸 Английский
- 🇪🇸 Испанский
- 🇵🇹 Португальский
- 🇸🇦 Арабский (фусха/MSA)
- 🇫🇷 Французский

### 📧 Интеграция с почтовыми клиентами
- **Gmail** - полная поддержка
- **Yandex Mail** - полная поддержка
- **Outlook** (outlook.live.com, outlook.office.com) - полная поддержка
- **Mail.ru** - полная поддержка

## 🚀 Установка

### Для разработки и тестирования:

1. **Клонируйте или скачайте проект**
   ```bash
   git clone <repository-url>
   cd email_enhancer_extension
   ```

2. **Откройте Chrome и перейдите в настройки расширений**
   ```
   chrome://extensions/
   ```

3. **Включите режим разработчика**
   - Переключите тумблер "Режим разработчика" в правом верхнем углу

4. **Загрузите расширение**
   - Нажмите "Загрузить распакованное расширение"
   - Выберите папку с файлами расширения

5. **Проверьте работу**
   - Откройте `test.html` для тестирования функций
   - Перейдите в любой поддерживаемый почтовый клиент

## 🎯 Использование

### Основной интерфейс

После установки расширения при открытии поля ввода в поддерживаемых почтовых клиентах появится панель инструментов Email Enhancer с следующими элементами:

1. **🎤 Кнопка записи голоса** - для голосового ввода
2. **✅ Кнопка исправления** - для автоматического исправления ошибок
3. **🤖 Кнопка AI-улучшения** - с выпадающим меню стилей
4. **🌍 Селектор языка** - для выбора языка обработки
5. **⚙️ Кнопка настроек** - для открытия страницы настроек

### Работа с текстом

1. **Выделение текста**: Выделите часть текста для применения функций только к выделенному фрагменту
2. **Обработка всего текста**: Если текст не выделен, функции применяются ко всему содержимому поля
3. **Голосовой ввод**: Нажмите на микрофон, разрешите доступ и начните говорить

### Настройки

Откройте страницу настроек для:
- Выбора языка интерфейса
- Установки языка обработки по умолчанию
- Настройки стиля улучшения по умолчанию
- Управления уведомлениями
- Настройки дополнительных функций

## 🔧 Техническая информация

### Архитектура

Расширение построено на основе Manifest V3 и состоит из:

- **Content Script** (`content.js`) - внедряется в страницы почтовых клиентов
- **Background Script** (`background.js`) - обрабатывает API запросы и управляет состоянием
- **Popup** (`popup.html/js`) - быстрый доступ к функциям
- **Options Page** (`options.html/js`) - полная страница настроек

### Интеграция с API

Расширение готово для интеграции с API `https://app.base44.com` через следующие эндпоинты:

- `POST /api/transcribe` - преобразование речи в текст
- `POST /api/fix-text` - исправление ошибок в тексте
- `POST /api/improve-text` - улучшение текста с помощью AI
- `GET /api/subscription/status` - проверка статуса подписки

### Безопасность

- Все данные обрабатываются через защищенные HTTPS соединения
- Локальное хранение настроек пользователя
- Минимальные разрешения для работы расширения

## 🔄 Разработка и вклад

### Добавление нового почтового клиента

1. Создайте новый класс провайдера в `js/email-providers.js`
2. Определите селекторы для полей ввода и кнопок
3. Добавьте домен в `manifest.json`

### Добавление нового языка

1. Добавьте переводы в `js/translations.js`
2. Обновите список языков в настройках
3. Добавьте флаг языка в соответствующие функции

### Стилизация

- Основные стили в `styles/content.css`
- Стили настроек в `styles/options.css`
- Поддержка темной темы через CSS медиа-запросы
