📦 Как установить расширение для тестирования

  1. Откройте Chrome и перейдите в chrome://extensions/
  2. Включите "Режим разработчика" в правом верхнем углу
  3. Нажмите "Загрузить распакованное расширение"
  4. Выберите папку с файлами расширения
  5. Обновите эту страницу для проверки работы

1 Тест основных функций

Инструкции по тестированию:

  • Введите текст в поле ниже
  • Используйте панель инструментов Email Enhancer (должна появиться под полем)
  • Протестируйте функции записи голоса, исправления и улучшения текста
  • Попробуйте выделить часть текста и применить функции только к выделенному

2 Тест с предзаполненным текстом

Тестирование исправления ошибок:

  • В поле ниже есть текст с ошибками
  • Используйте функцию "Исправить текст" для автоматического исправления
  • Попробуйте разные стили улучшения текста
Привет! Как дела? Я хотел бы обсудить с вами важный вопрос по поводу нашего проекта. Можем ли мы встретится завтра в 15:00? Буду ждать вашего ответа.

3 Тест многоязычности

Тестирование языковых функций:

  • Измените язык обработки текста в панели инструментов
  • Введите текст на разных языках
  • Проверьте работу функций с разными языками
Hello! How are you doing today? I hope everything is going well.

🎤 Голосовой ввод

Записывайте голосовые сообщения и автоматически преобразуйте их в текст

Нажмите на иконку микрофона и начните говорить. Текст будет автоматически вставлен в поле ввода.

✅ Исправление ошибок

Автоматическое исправление орфографических, грамматических и пунктуационных ошибок

Выделите текст с ошибками и нажмите кнопку исправления для автоматической коррекции.

🤖 AI-улучшение

Улучшение текста с помощью искусственного интеллекта в разных стилях

Выберите стиль (дружелюбный, официальный, краткий, убеждающий, технический) и улучшите текст.

🌍 Многоязычность

Поддержка 6 языков: русский, английский, испанский, португальский, арабский, французский

Переключайте языки обработки и интерфейса для работы с международной корреспонденцией.

⚙️ Настройки

Гибкая настройка функций, языков и поведения расширения

Откройте настройки для персонализации работы расширения под ваши потребности.

📧 Интеграция

Работает с Gmail, Yandex Mail, Outlook, Mail.ru

Расширение автоматически интегрируется с популярными почтовыми клиентами.
